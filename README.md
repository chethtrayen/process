# Process

## Concept

### Ideation

### Decision 
  - Decide core feature
    - User Experience
    - Use case
  - Building requirements

### Design

## POC (Hackathon)
  - Length: <1-3 weeks
  - Limited stack
    - Frontend (optional)
    - Backend (optional)
    - Docker database (optional)
  - Test new tech, lib, framework, etc.
  - One small repo

  - Defined code logic
  - Finish core feature

- Build production code
- Deploy on a server
- Remote database
